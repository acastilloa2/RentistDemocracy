\section{📄 Description}\label{description}

This repository contains the replication files for the project
\emph{``Rentist Democracy: Distributive Institutions and Political
Competition in Colombian Local Elections''}. The analysis explores the
impact of oil and gas royalties on political competition using panel
data from Colombian municipalities.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{📁 Folder Structure}\label{folder-structure}

\begin{itemize}
\tightlist
\item
  \texttt{Data/}: Raw and cleaned datasets.\\
\item
  \texttt{Outputs/}: Tables and plots generated by the analysis.\\
\item
  \texttt{Scripts/}: R scripts organized by function.\\
\item
  \texttt{README.md}: This documentation file.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{🧪 Replication Steps}\label{replication-steps}

To replicate the analysis:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Open \texttt{Scripts/00\_Master\_Script.R} in RStudio.\\
\item
  Run the script. It will:

  \begin{itemize}
  \tightlist
  \item
    Clean and reshape ANH data.\\
  \item
    Clean and harmonize IICA conflict data.\\
  \item
    Merge CEDE panel data with conflict and production data.\\
  \item
    Generate regression models and export results.
  \end{itemize}
\end{enumerate}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}
